# My First React Project - Personal Portfolio

Hello! 👋 Welcome to my very first React/Next.js project! I created this personal portfolio to showcase my skills, projects, and the things I am learning as a beginner developer.

🌍 **Live Demo:** [Check out my live portfolio here!](https://portfolio-eight-beta-hkm4cerpar.vercel.app/)

## About This Project
This website is a personal portfolio that I built to practice modern web development. It features smooth animations, a modern responsive design, and organized code structure. It was built using:
- **Next.js & React.js** (for building the user interface)
- **Tailwind CSS** (for styling and making it look good on all devices)
- **Framer Motion** (for the cool smooth animations)

## What's Inside?
- **Hero Section:** A quick introduction.
- **About Me:** Information about my background and skills.
- **Services/Work:** A look into what I can build.
- **Contact:** A way to reach out to me!

## My Learning Journey
Since this is my very first React project, my main goal was to understand how to build responsive layouts, work with React components, and deploy a live website. I'm constantly learning and tweaking things as I go!
